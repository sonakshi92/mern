# Laundry Mart

Laundry Mart is a responsive laundry service website built using HTML and CSS.

## Features

- Fixed Navigation Bar
- Hero Section
- About Us
- Services
- Price List
- Booking Form
- Contact Section

## Technologies Used

- HTML5
- CSS3
- Flexbox

## Project Structure

Laundry-Mart/
│
├── index.html
├── css/
│   └── styles.css
├── images/
└── README.md

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click index.html

OR

Open the folder in VS Code and use Live Server.
